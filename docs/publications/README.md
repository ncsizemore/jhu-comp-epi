# Publication PDFs

This directory contains PDF copies of featured publications for reference during content creation (extracting key figures and key findings).

## Structure

Store PDFs with their publication ID as the filename:
- `althoff-2024.pdf`
- `hyle-2023.pdf`
- `althoff-2021.pdf`
- `fojo-2021-end-hiv.pdf`

## Usage

These PDFs are:
- Git tracked for team reference
- **NOT** included in the Vercel deployment bundle (see `.gitignore`)
- Used to extract key figures and verify key findings with authors
- For internal use only - respect copyright and licensing
